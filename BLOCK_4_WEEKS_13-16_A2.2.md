### **Monday: Conditionnel formation**

**Grammar (10min):**

**Conditionnel Présent = Would + verb**

**Formation:**
Futur stem + Imparfait endings

```
Futur stem: parler → parler-
Imparfait endings: -ais, -ais, -ait, -ions, -iez, -aient

je parlerais     I would speak
tu parlerais     you would speak
il/elle parlerait  he/she would speak
nous parlerions  we would speak
vous parleriez   you would speak
ils/elles parleraient  they would speak
```

**Same irregular stems as futur:**
```
être → ser-     je serais
avoir → aur-    j'aurais
aller → ir-     j'irais
faire → fer-    je ferais
pouvoir → pourr- je pourrais
vouloir → voudr- je voudrais
venir → viendr-  je viendrais
voir → verr-     je verrais
```

English comparison:
English: "would" + verb (I would speak)
French: One word (Je parlerais)
Same relationship as future: simple in French!

Examples:
1. J'aimerais un café, s'il vous plaît. (I would like a coffee, please.)
2. Tu pourrais m'aider? (Could you help me?)
3. Il voudrait partir. (He would like to leave.)
4. Nous aimerions voyager. (We would like to travel.)
5. Vous devriez étudier. (You should study.)
6. Ils feraient ça? (Would they do that?)

**Vocabulary (5min):**
1. **Aimer** [eme] (conditional) - Would like
   - *J'aimerais un café.* (I would like a coffee.)
   
2. **Pouvoir** [puvwaʁ] (conditional) - Could
   - *Tu pourrais m'aider?* (Could you help me?)
   
3. **Devoir** [dəvwaʁ] (conditional) - Should
   - *Vous devriez partir.* (You should leave.)

**Speaking (10min):**
Scenario: Polite requests
```
AI: Tu as besoin de quelque chose?
YOU: [Make polite requests]
Expected: Je voudrais... Pourriez-vous...? J'aimerais...
```

**Mini Quiz (5min):**
1. Form conditionnel: je + vouloir
2. Form conditionnel: nous + être
3. Translate: I would like

**Homework (Audio):**
Record 10 polite requests/wishes using conditionnel.

### **Tuesday: Conditionnel for politeness**

**Grammar (10min):**

**Conditionnel for Polite Requests:**

**Direct (rude):**
- Je veux un café. (I want a coffee.)
- Donne-moi ça! (Give me that!)

**Polite (conditionnel):**
- Je voudrais un café. (I would like a coffee.)
- Pourrais-tu me donner ça? (Could you give me that?)

**Super polite formulas:**
- Je voudrais... (I would like...)
- J'aimerais... (I would like...)
- Pourriez-vous...? (Could you...?)
- Auriez-vous...? (Would you have...?)
- Est-ce que je pourrais...? (Could I...?)

**With "devoir" = should/ought to:**
- Tu devrais étudier. (You should study.)
- Nous devrions partir. (We should leave.)
- Vous devriez voir un médecin. (You should see a doctor.)

English comparison:
English: "would like" is polite, "want" is direct
French: same distinction with conditionnel vs présent
Universal politeness pattern!

Examples:
1. Pourriez-vous m'aider, s'il vous plaît? (Could you help me, please?)
2. J'aimerais réserver une table. (I would like to reserve a table.)
3. Auriez-vous l'heure? (Would you have the time?)
4. Tu devrais te reposer. (You should rest.)
5. Nous voudrions trois billets. (We would like three tickets.)
6. Est-ce que je pourrais essayer? (Could I try?)

**Vocabulary (5min):**
1. **Réserver** [ʁezɛʁve] - To reserve
   - *Je voudrais réserver une chambre.* (I would like to reserve a room.)
   
2. **Essayer** [eseje] - To try
   - *Est-ce que je pourrais essayer?* (Could I try?)
   
3. **Se reposer** [sərəpoze] - To rest
   - *Tu devrais te reposer.* (You should rest.)

### **Wednesday: Conditionnel with "si" clauses**

**Grammar (10min):**

**Si + Imparfait + Conditionnel = If...would...**

Pattern: Si + [imparfait], [conditionnel]

**Structure:**
```
Si j'avais de l'argent, j'achèterais une maison.
(If I had money, I would buy a house.)

Si clause (condition) → Imparfait
Main clause (result) → Conditionnel
```

**Can reverse order:**
- J'achèterais une maison si j'avais de l'argent.

**Common formulas:**
- Si j'étais toi, je... (If I were you, I would...)
- Si je pouvais, je... (If I could, I would...)
- Si tu voulais, tu... (If you wanted, you would/could...)

English comparison:
English: "If I had...I would..."
French: "Si j'avais...je..." (EXACT parallel!)
One of the easiest French structures for English speakers!

**CRITICAL:** Never use conditionnel after "si"!
❌ Si je serais riche... 
✅ Si j'étais riche...

Examples:
1. Si j'avais le temps, je voyagerais. (If I had time, I would travel.)
2. Si tu étudiais plus, tu réussirais. (If you studied more, you would succeed.)
3. Si nous habitions à Paris, nous visiterions les musées. (If we lived in Paris, we would visit museums.)
4. Qu'est-ce que tu ferais si tu gagnais à la loterie? (What would you do if you won the lottery?)
5. Si j'étais toi, je partirais. (If I were you, I would leave.)
6. Elle viendrait si elle pouvait. (She would come if she could.)

**Vocabulary (5min):**
1. **Gagner** [ɡaɲe] - To win/earn
   - *Si je gagnais à la loterie...* (If I won the lottery...)
   
2. **Réussir** [ʁeysiʁ] - To succeed
   - *Si tu étudiais, tu réussirais.* (If you studied, you would succeed.)
   
3. **Loterie** [lɔtʁi] - Lottery
   - *J'achèterais une maison si je gagnais à la loterie.* (I would buy a house if I won the lottery.)

**Speaking (10min):**
Scenario: Hypothetical questions
```
AI: Si tu avais un million d'euros, que ferais-tu?
YOU: [Describe hypothetical actions]
Expected: Si j'avais un million, j'achèterais... Je voyagerais...
```

**Mini Quiz (5min):**

2. Fill: Si tu avais le temps, tu ____? (a) viendrais (b) venais
3. Translate: If I were you, I would leave

### **Thursday: Conditionnel for advice & suggestions**

**Grammar (10min):**

**Conditionnel for Advice:**

**Tu devrais... / Vous devriez... = You should...**
- Tu devrais voir un médecin. (You should see a doctor.)
- Vous devriez vous reposer. (You should rest.)

**Il faudrait... = One should/It would be necessary...**
- Il faudrait partir tôt. (One should leave early.)
- Il faudrait réserver. (One should reserve.)

**On pourrait... = We/One could...**
- On pourrait aller au cinéma. (We could go to the cinema.)
- On pourrait essayer. (We could try.)

**À ta/votre place, je... = In your place, I would...**
- À ta place, je partirais. (In your place, I would leave.)
- À votre place, j'accepterais. (In your place, I would accept.)

English comparison:
English: "should," "ought to," "could"
French: conditionnel of devoir, pouvoir, falloir
Modal verbs + conditionnel = softened advice

Examples:
1. Tu devrais manger plus de légumes. (You should eat more vegetables.)
2. Il faudrait étudier tous les jours. (One should study every day.)
3. On pourrait partir demain. (We could leave tomorrow.)
4. À ta place, je refuserais. (In your place, I would refuse.)
5. Vous devriez apprendre le français. (You should learn French.)
6. On pourrait inviter Marie. (We could invite Marie.)

**Vocabulary (5min):**
1. **Accepter** [aksɛpte] - To accept
   - *Tu devrais accepter cette offre.* (You should accept this offer.)
   
2. **Refuser** [ʁəfyze] - To refuse
   - *Je refuserais à ta place.* (I would refuse in your place.)
   
3. **Inviter** [ɛ̃vite] - To invite
   - *On pourrait inviter nos amis.* (We could invite our friends.)

**Speaking (10min):**
Scenario: Give advice
```
AI presents problems: "Je suis fatigué," "Je n'ai pas d'argent," etc.

Expected: Tu devrais... À ta place, je... On pourrait...
```

**Mini Quiz (5min):**

2. Translate: We could try

### **Monday: Pronoun Y (there/to it)**

**Grammar (10min):**

**Pronoun Y replaces:**

**1. Places (à/en/dans + place):**
- Tu vas à Paris? → Oui, j'**y** vais. (Yes, I'm going there.)
- Il habite en France? → Oui, il **y** habite. (Yes, he lives there.)

**2. Things after "à":**
- Tu penses à l'examen? → Oui, j'**y** pense. (Yes, I'm thinking about it.)
- Elle répond aux questions? → Oui, elle **y** répond. (Yes, she answers them.)

**Position:** Before the verb (same as direct/indirect pronouns)
- Je vais à Paris. → J'**y** vais.
- Nous pensons au problème. → Nous **y** pensons.

**With passé composé:** Before auxiliary
- Je suis allé à Paris. → J'**y** suis allé.

**With futur proche/modal:** Before infinitive
- Je vais aller à Paris. → Je vais **y** aller.
- Je peux aller à Paris. → Je peux **y** aller.

English comparison:
English: "there" or "about it"
French: **y** covers both!
Position is different (before verb in French)

Examples:
1. Tu vas au cinéma? - Oui, j'y vais. (Are you going to the cinema? - Yes, I'm going there.)
2. Elle habite en France? - Non, elle n'y habite pas. (Does she live in France? - No, she doesn't live there.)
3. Tu penses au problème? - Oui, j'y pense. (Are you thinking about the problem? - Yes, I'm thinking about it.)
4. Nous sommes allés à Paris. - Nous y sommes allés hier. (We went to Paris. - We went there yesterday.)
5. On peut aller au parc? - Oui, on peut y aller. (Can we go to the park? - Yes, we can go there.)
6. Vous vous intéressez à la politique? - Non, je ne m'y intéresse pas. (Are you interested in politics? - No, I'm not interested in it.)

**Vocabulary (5min):**
1. **Penser à** [pɑ̃se] - To think about
   - *Tu penses à l'examen? - Oui, j'y pense.* (Are you thinking about the exam? - Yes, I'm thinking about it.)
   
2. **S'intéresser à** [sɛ̃teʁese] - To be interested in
   - *Elle s'intéresse à l'art? - Oui, elle s'y intéresse.* (Is she interested in art? - Yes, she's interested in it.)
   
3. **Répondre à** [ʁepɔ̃dʁ] - To answer
   - *Tu réponds à la question? - Oui, j'y réponds.* (Are you answering the question? - Yes, I'm answering it.)

**Speaking (10min):**
Scenario: Q&A with Y
```
AI: Tu vas souvent au cinéma? Tu habites en ville? etc.
YOU: Answer using Y
Expected: Oui, j'y vais souvent. Non, je n'y habite pas.
```

**Mini Quiz (5min):**
1. Replace: Je vais à Paris. → ________
2. Replace: Elle pense au problème. → ________
3. Position of Y with passé composé?

**Homework (Text):**
Write 10 Q&A about places and abstract things, replacing with Y.

### **Tuesday: Pronoun EN (of it/some)**

**Grammar (10min):**

**Pronoun EN replaces:**

**1. Quantities (de + noun):**
- Tu veux du café? → Oui, j'**en** veux. (Yes, I want some.)
- Il a des frères? → Oui, il **en** a trois. (Yes, he has three of them.)

**2. Things after "de":**
- Tu parles de ton travail? → Oui, j'**en** parle. (Yes, I'm talking about it.)
- Elle a besoin d'aide? → Oui, elle **en** a besoin. (Yes, she needs it.)

**3. Partitive articles (du/de la/des):**
- Tu manges de la viande? → Non, je n'**en** mange pas. (No, I don't eat any.)

**4. With numbers (keep the number):**
- Tu as combien de frères? → J'**en** ai deux. (I have two of them.)

**Position:** Same as Y (before verb)

English comparison:
English: "some," "any," "of it/them"
French: **en** (one word!)
Often omitted in English but required in French

Examples:
1. Tu veux du pain? - Oui, j'en veux. (Do you want bread? - Yes, I want some.)
2. Il a des enfants? - Oui, il en a trois. (Does he have children? - Yes, he has three.)
3. Tu parles de tes problèmes? - Non, je n'en parle pas. (Do you talk about your problems? - No, I don't talk about them.)
4. Elle a besoin d'argent? - Oui, elle en a besoin. (Does she need money? - Yes, she needs it.)
5. Vous avez mangé des fruits? - Oui, nous en avons mangé. (Did you eat fruit? - Yes, we ate some.)
6. Combien de livres as-tu? - J'en ai beaucoup. (How many books do you have? - I have many.)

**Vocabulary (5min):**
1. **Avoir besoin de** [avwaʁbəzwɛ̃] - To need
   - *Tu as besoin d'aide? - Oui, j'en ai besoin.* (Do you need help? - Yes, I need it.)
   
2. **Parler de** [paʁle] - To talk about
   - *On parle du film? - Oui, on en parle.* (Are we talking about the movie? - Yes, we're talking about it.)
   
3. **Viande** [vjɑ̃d] - Meat
   - *Tu manges de la viande? - Non, je n'en mange pas.* (Do you eat meat? - No, I don't eat any.)

**Speaking (10min):**
Scenario: Shopping/quantity questions
```

```

### **Wednesday: Y and EN together**

**Grammar (10min):**

**Using Both Y and EN:**

**Different contexts:**
- Y for places/à + thing
- EN for quantities/de + thing

**Examples:**
- Je vais à Paris. → J'**y** vais.
- Je viens de Paris. → J'**en** viens. (I come from there.)
- Il y a du pain? → Oui, il **y** **en** a. (Yes, there is some.)

**Il y en a = There is/are some:**
```
Il y a du pain? - Oui, il y en a.
Il y a des pommes? - Oui, il y en a trois.
```

**Order with two pronouns:**
Y + EN when both used (rare):
- Il y en a. (There is/are some.)

**Common verbs with Y:**
- aller à, penser à, répondre à, s'intéresser à

**Common verbs with EN:**
- avoir besoin de, parler de, venir de, avoir de

English comparison:
English often omits these
French REQUIRES them
"Do you have any?" → "Tu en as?"
(Can't just say "Tu as?")

Examples:
1. Tu vas au marché? Il y a des fruits? - Oui, j'y vais et oui, il y en a. (Are you going to the market? Are there fruits? - Yes, I'm going there and yes, there are some.)
2. Tu viens de Paris? - Oui, j'en viens. (Are you coming from Paris? - Yes, I'm coming from there.)
3. Il y a du lait dans le frigo? - Oui, il y en a. (Is there milk in the fridge? - Yes, there is some.)
4. Tu penses à ton examen et tu parles de tes notes? - Oui, j'y pense et j'en parle. (Are you thinking about your exam and talking about your grades? - Yes, I'm thinking about it and talking about them.)

**Vocabulary (5min):**
1. **Marché** [maʁʃe] - Market
   - *Tu vas au marché? - Oui, j'y vais.* (Are you going to the market? - Yes, I'm going there.)
   
2. **Venir de** [vəniʁ] - To come from
   - *Tu viens de France? - Oui, j'en viens.* (Are you coming from France? - Yes, I'm coming from there.)
   
3. **Frigo** [fʁigo] - Fridge (informal)
   - *Il y a du lait dans le frigo? - Oui, il y en a.* (Is there milk in the fridge? - Yes, there is some.)

### **Monday: QUI (who/that/which - subject)**

**Grammar (10min):**

**Relative Pronoun QUI = who/that/which (SUBJECT)**

**Function:** Replaces subject of relative clause

**Pattern:**
Noun + QUI + verb

**Examples:**
- C'est l'homme. L'homme parle. → C'est l'homme **qui** parle. (It's the man who is speaking.)
- J'ai un ami. Cet ami habite à Paris. → J'ai un ami **qui** habite à Paris. (I have a friend who lives in Paris.)

**QUI for people and things:**
- La fille qui danse. (The girl who is dancing.)
- Le livre qui est sur la table. (The book that is on the table.)

**Verb agreement:**
QUI + verb agrees with antecedent (the noun QUI replaces)
- C'est moi qui **suis** professeur. (It's me who is a teacher.)
- C'est toi qui **as** raison. (It's you who is right.)

English comparison:
English: "who" (people), "that/which" (things)
French: QUI (both people AND things!)
Simpler than English in this way

Examples:
1. C'est la fille qui parle français. (It's the girl who speaks French.)
2. J'aime les films qui font rire. (I like movies that make you laugh.)
3. Où est le livre qui était ici? (Where is the book that was here?)
4. C'est nous qui avons gagné. (We're the ones who won. / It's us who won.)
5. Les étudiants qui travaillent réussissent. (Students who work succeed.)
6. La maison qui est à droite est belle. (The house that is on the right is beautiful.)

**Vocabulary (5min):**
1. **Faire rire** [fɛʁʁiʁ] - To make laugh
   - *J'aime les films qui font rire.* (I like movies that make you laugh.)
   
2. **Gagner** [ɡaɲe] - To win
   - *C'est nous qui avons gagné.* (We're the ones who won.)
   
3. **Réussir** [ʁeysiʁ] - To succeed
   - *Les étudiants qui travaillent réussissent.* (Students who work succeed.)

**Speaking (10min):**
Scenario: Describe people and things
```

YOU: Create complex descriptions
Expected: C'est l'homme qui... J'aime les livres qui...
```

**Mini Quiz (5min):**
1. Combine: C'est la fille. Elle parle français. → ________
2. Fill: C'est moi qui ____ professeur. (a) suis (b) es (c) est
3. Translate: The book that is on the table

### **Tuesday: QUE (whom/that/which - object)**

**Grammar (10min):**

**Relative Pronoun QUE = whom/that/which (OBJECT)**

**Function:** Replaces direct object of relative clause

**Pattern:**
Noun + QUE + subject + verb

**Examples:**
- C'est le film. J'ai vu le film. → C'est le film **que** j'ai vu. (It's the movie that I saw.)
- La fille. Tu connais la fille. → La fille **que** tu connais. (The girl whom you know.)

**Before vowel: QU'**
- L'homme qu'elle aime. (The man whom she loves.)

**Key difference from QUI:**
- QUI + verb (subject)
- QUE + subject + verb (object)

English comparison:
English: "whom/that" (often omitted: "the movie I saw")
French: QUE is REQUIRED (never omitted!)
"The movie I saw" → ❌ in French
"Le film que j'ai vu" → ✅ Must include QUE

Examples:
1. C'est le livre que je lis. (It's the book that I'm reading.)
2. La fille que tu connais habite ici. (The girl whom you know lives here.)
3. Les films que j'aime sont français. (The movies that I like are French.)
4. C'est l'homme qu'elle a rencontré. (It's the man whom she met.)
5. Voici le cadeau que j'ai acheté. (Here's the gift that I bought.)
6. La voiture que nous avons vue est chère. (The car that we saw is expensive.)

**Vocabulary (5min):**
1. **Rencontrer** [ʁɑ̃kɔ̃tʁe] - To meet
   - *C'est l'homme que j'ai rencontré hier.* (It's the man whom I met yesterday.)
   
2. **Cadeau** [kado] - Gift
   - *Le cadeau que j'ai reçu est beau.* (The gift that I received is beautiful.)
   
3. **Recevoir** [ʁəsəvwaʁ] - To receive
   - *J'aime les lettres que je reçois.* (I like the letters that I receive.)

**Speaking (10min):**
Scenario: Describe things you like/know/saw
```
AI prompts: What movies do you like? Who do you know? etc.
YOU: Use QUE
Expected: Les films que j'aime... Les gens que je connais...
```

**Mini Quiz (5min):**
1. Combine: C'est le film. J'ai vu le film. → ________
2. Why QU': L'homme __elle aime. (a) que (b) qu'

### **Wednesday: QUI vs QUE distinction**

**Grammar (10min):**

**How to Choose: QUI or QUE?**

**Test: What follows?**

**If VERB directly follows → QUI (subject)**
- L'homme **qui** parle. (The man who speaks.)
- Les livres **qui** sont ici. (The books that are here.)

**If SUBJECT + VERB follow → QUE (object)**
- L'homme **que** je connais. (The man whom I know.)
- Les livres **que** j'ai lus. (The books that I read.)

**Mnemonic:**
- QUI = QUI + Verb
- QUE = QUE + sUbject + vErb

**Side by side:**
- C'est le professeur **qui** enseigne le français. (subject: professeur teaches)
- C'est le professeur **que** j'aime beaucoup. (object: I like professeur)

English comparison:
English: Sometimes drops relative pronoun
French: NEVER drops QUI or QUE
"The movie I saw" → Must be "Le film que j'ai vu"

Examples:
1. La fille **qui** chante est ma sœur. (subject)
2. La fille **que** tu vois est ma sœur. (object)
3. Le livre **qui** est sur la table est intéressant. (subject)
4. Le livre **que** je lis est intéressant. (object)
5. Les étudiants **qui** travaillent réussissent. (subject)
6. Les étudiants **que** je connais sont gentils. (object)

**Vocabulary (5min):**
1. **Enseigner** [ɑ̃seɲe] - To teach
   - *Le professeur qui enseigne est excellent.* (The teacher who teaches is excellent.)
   
2. **Chanter** [ʃɑ̃te] - To sing
   - *La chanson qu'elle chante est belle.* (The song that she sings is beautiful.)
   
3. **Chanson** [ʃɑ̃sɔ̃] - Song
   - *C'est la chanson que j'aime.* (It's the song that I like.)

### **Monday: Reflexive verbs - reciprocal**

**Grammar (10min):**

**Reflexive = action on oneself**
**Reciprocal = action on each other**

**Same form, different meaning:**

**Reflexive:**
- Ils se lavent. (They wash themselves.)
- Nous nous parlons. (We talk to ourselves.)

**Reciprocal:**
- Ils se lavent. (They wash each other.) - could also mean this!
- Nous nous parlons. (We talk to each other.)

**Context determines meaning!**

**Common reciprocal verbs:**
- se parler (to talk to each other)
- se voir (to see each other)
- se téléphoner (to call each other)
- se rencontrer (to meet each other)
- se connaître (to know each other)
- s'aimer (to love each other)
- se quitter (to leave each other)
- se disputer (to argue with each other)

**To clarify reciprocal:**
Add "l'un l'autre" (each other):
- Ils se parlent l'un à l'autre. (They talk to each other.)

English comparison:
English: "each other" makes it clear
French: context or add "l'un l'autre"
Same verb form for both uses!

Examples:
1. Marie et Paul se parlent tous les jours. (Marie and Paul talk to each other every day.)
2. Nous nous voyons souvent. (We see each other often.)
3. Ils se sont rencontrés en 2020. (They met each other in 2020.)
4. Vous vous connaissez? (Do you know each other?)
5. Elles s'aiment beaucoup. (They love each other very much.)
6. Nous nous téléphonons chaque semaine. (We call each other every week.)

**Vocabulary (5min):**
1. **Se voir** [səvwaʁ] - To see each other
   - *On se voit demain?* (Shall we see each other tomorrow?)
   
2. **Se téléphoner** [sətelefoˈne] - To call each other
   - *Nous nous téléphonons souvent.* (We call each other often.)
   
3. **Se disputer** [sədispyte] - To argue
   - *Ils se disputent tout le temps.* (They argue all the time.)


1. Translate: We see each other often
2. Translate: They love each other
3. Which means reciprocal: se parler (alone) or needs context?
