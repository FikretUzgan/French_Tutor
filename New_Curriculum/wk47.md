### Scaffolding Steps

**Structured Debate in French:**

**1. DEBATE FORMAT:**

```
PROPOSITION: "L'éducation uniforme produit des citoyens plus égalitaires."

PRO SPEAKER (opening): 3 minutes
- Present main arguments
- Provide evidence

CON SPEAKER (opening): 3 minutes
- Counter the proposition
- Present alternative arguments

PRO RESPONSE: 2 minutes
- Address counterarguments
- Strengthen position

CON RESPONSE: 2 minutes
- Address pro response
- Conclude con position

AUDIENCE Q&A: Variable
```

**2. DEBATE ARGUMENT CONSTRUCTION:**

```
STRONG ARGUMENT STRUCTURE:
1. Claim: "Éducation uniforme produit inégalité, non égalité."
2. Evidence: "Études montrent que les étudiants avec besoins spécialisés échouent davantage..."
3. Reasoning: "Parce qu'un curriculum unique ignore la diversité cognitive..."
4. Impact: "...ce qui perpétue un cycle d'inégalité."

WEAK ARGUMENT:
"L'éducation uniforme, c'est pas bon parce que c'est nul."
[No evidence, no reasoning, emotional only]

STRONGER:
"Tandis que l'uniformité prétend créer l'égalité, les données empiriques révèlent le contraire: 
les systèmes scolaires les plus uniformes (Singapour, Japon) produisent aussi les écarts d'achievement 
les plus importants. Ceci démontre que l'égalité requiert une différenciation pédagogique, non son absence."
```

**3. RESPONDING TO COUNTERARGUMENTS:**

```
ATTACK THE ARGUMENT NOT THE PERSON:
WEAK: "Mon adversaire est stupide."
STRONG: "L'argument de mon adversaire repose sur l'assomption que... Cependant, les faits contredisent..."

CONCEDE PARTIAL TRUTH:
"Mon adversaire soulève un point valide concernant les coûts économiques. Néanmoins,
cet argument ne réfute pas le point central que l'égalité requiert...

SHIFT THE BURDEN:
"Mon adversaire affirme que X. Mais où sont les preuves? Tant qu'il ne fournit pas de données,
mon argument reste plus solide."

REVEAL UNSTATED ASSUMPTIONS:
"Cet argument présume que les gens sont rationnels et économiquement motivés. Or, la psychologie
comportementale démontre..."
```

**4. DEBATE LANGUAGE MARKERS:**

```
OPENING CLAIM:
"Je soutiens que..."
"L'argument pro (ou con) est que..."
"Ma position s'appuie sur trois pilliers:"

INTRODUCING EVIDENCE:
"Comme l'a démontré [expert]..."
"Les données de [source] révèlent..."
"Un exemple parlant est celui de..."

CONCEDING POINTS:
"Je reconnais que mon adversaire soulève..."
"C'est un point important, et je l'accorde partiellement."

REFUTING:
"Au contraire, les faits contredisent..."
"Mon adversaire confond cause et corrélation..."
"Cet argument enfreint la logique de base car..."

STRENGTHENING:
"Cela renforce mon argument car..."
"C'est particulièrement problématique à la lumière de..."
```

## Speaking Scenario - Structured Debate
**Domain:** Formal debate on substantive topic

**PROPOSITION:** "Les universités devraient abolir les examens traditionnels."

**PRO OPENING (3 min outline):**
- Claim: Tests poorly measure learning
- Evidence: Psychological research on test anxiety
- Example: Countries with alternative assessment (project-based) outperform test-heavy systems
- Impact: Abolishing tests would improve equity and genuine learning

**CON OPENING (3 min outline):**
- Counterclaim: Some standardization necessary
- Evidence: Chaos without common benchmarks
- Example: Countries that tried all project-based learning showed accountability gaps
- Impact: Complete abolition would harm educational quality assurance

## Homework (Audio) - Participate in a Structured Debate

You will debate a proposition with an AI opponent (or recording partner).

**Your assignment:**
1. Choose a proposition (or use provided one)
2. Choose a position (Pro or Con)
3. Prepare and deliver:
   - **Opening statement:** 3 minutes
     - State your position clearly
     - Present 2-3 main arguments
     - Provide at least 1 concrete example or statistic
   
   - **Response:** 2 minutes
     - Address main counterarguments
     - Strengthen your position
     - Maintain logical flow

**Language Requirements:**
- Use formal debate register throughout
- Include at least 3 different argument introduction phrases
- Concede partial truth where appropriate
- Use subjunctive for hypotheticals
- Avoid ad hominem attacks
- Support claims with evidence/reasoning

**Proposition Options:**
- Remote work should be permanent default for office jobs
- Universities should prioritize vocational training over pure academics
- AI should be heavily regulated for safety
- Universal basic income would improve society
- Social media should be banned for under-18s

Rubric:
- Argument quality & logic (30%)
- Evidence & supporting details (25%)
- Responsiveness to opposition (20%)
- Language accuracy & debate register (15%)
- Clarity & persuasiveness (10%)
