## Week 24 / Block 6 / CEFR Level B1.2

### Learning Outcomes (CEFR Descriptors)
- Can produce coherent, well-structured argumentative or explanatory texts of 180–250 words with clear progression, appropriate connectors and formal/semi-formal register.
- Can write a short formal letter or email (complaint, application, request) following conventional French structure and politeness formulas.
- Can self-assess and correct own production using B1.2-level criteria.

### Interaction Tier
**Tier:** 2
**Interaction Type:** Semi-Initiated

### Grammar Target
**Form:** Synthèse B1.2 – révision intégrée (subjonctif, plus-que-parfait, voix passive, passé simple recognition, conjonctions complexes, marqueurs de discours)
**Complexity Rating:** 8/10
**Prerequisite Knowledge:** Toutes les structures vues dans Block 5 et Block 6 jusqu’ici
**Scaffolding Steps:**
1. Diagnostic rapide : identifier les points faibles personnels dans les structures B1 (subjonctif, passif, connecteurs…)
2. Révision ciblée : 2–3 exercices mixtes par structure (transformation, gap-fill, correction)
3. Focus sur la cohérence textuelle : paragraphe introductif, développement argumenté, conclusion équilibrée
4. Structure de lettre/e-mail formelle : formule d’appel, objet, corps (3–4 paragraphes), formule de politesse
5. Production guidée : plan détaillé avant rédaction
6. Auto-correction avec grille B1.2 (cohérence, précision grammaticale, richesse lexicale, registre)
7. Production finale : texte complet 180–250 mots

**Structures prioritaires à réviser :**
- Subjonctif présent/passé après déclencheurs variés
- Plus-que-parfait pour antériorité
- Voix passive dans registres formels
- Passé simple (reconnaissance + 1–2 formes actives simples)
- Connecteurs et marqueurs de discours (≥10 par texte)

### Speaking Scenario
**Domain:** Simulation d’entretien ou de présentation formelle
Imagine que tu présentes oralement une réclamation ou une demande (stage, remboursement, explication d’un retard…).
Structure ton discours (introduction – problème – arguments – conclusion) et utilise au moins 8 connecteurs/marqueurs différents.

### Reading/Listening Component
**Type:** Lettre formelle + réponse + audio (160–200 secondes)
Lettre de plainte (produit défectueux) + réponse du service client
→ 10 questions (identifier connecteurs, reformuler en passif, vrai/faux, conjuguer au subjonctif/passé composé/plus-que-parfait, corriger erreur)

### Homework Assignment
**Type:** Texte
Rédige une lettre ou un e-mail formel (200–250 mots) :
- soit une réclamation détaillée (produit/service défectueux, retard de livraison, annulation injustifiée)
- soit une demande formelle (stage, information complémentaire, rendez-vous professionnel)
Obligatoire :
- ≥ 10 connecteurs/marqueurs de discours variés
- ≥ 3 structures complexes (subjonctif, passif, plus-que-parfait)
- Structure complète de lettre/e-mail française

**Rubric:**
- [ ] Structure formelle respectée (appel, objet, paragraphes clairs, politesse finale)
- [ ] ≥ 10 connecteurs/marqueurs utilisés correctement
- [ ] ≥ 3 structures complexes (subjonctif / passif / plus-que-parfait) bien employées
- [ ] ≥ 15 mots de la liste intégrés
- [ ] Texte cohérent, logique, registre soutenu B1.2
**Pass:** 4/5 critères remplis

### Assessment Checkpoint
**Mini-Quiz (8 questions):**
1. Complète : ……… il soit tard, je continue à travailler. → Bien que
2. Passif : Les documents ……… (envoyer) hier. → ont été envoyés
3. Plus-que-parfait : Quand je suis arrivé, le train ……… déjà ……… . → était / parti
4. Subjonctif passé : Je regrette que tu ……… (ne pas venir). → n’aies pas été venu(e)
5. Connecteur : Il pleut. ……… nous avons annulé la sortie. → Par conséquent
6. Traduis : “Je vous serais reconnaissant de bien vouloir m’envoyer le document.” → Je vous serais reconnaissant de bien vouloir m’envoyer le document.
7. Reformulation : Il est fatigué. → En d’autres termes, il dort mal.
8. Formule finale : Veuillez agréer, Monsieur, ……… de mes salutations distinguées. → l’expression
