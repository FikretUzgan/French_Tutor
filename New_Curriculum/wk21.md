## Week 21 / Block 6 / CEFR Level B1.2

### Learning Outcomes (CEFR Descriptors)
- Can use the passive voice to focus on the action or the receiver rather than the doer in formal, journalistic or neutral contexts.
- Can transform active sentences into passive in various tenses (présent, passé composé, futur simple, imparfait).
- Can understand passive constructions in news articles, instructions and formal texts.

### Interaction Tier
**Tier:** 2
**Interaction Type:** Semi-Initiated

### Grammar Target
**Form:** Voix passive – formation + usage + agent (par / de) + omission de l’agent
**Complexity Rating:** 7/10
**Prerequisite Knowledge:** Être + participe passé accord, tous les temps composés de base, subjonctif présent
**Scaffolding Steps:**
1. Rappel : voix passive = être (au temps voulu) + participe passé (accord avec sujet)
2. Formation dans les temps clés : présent, passé composé, futur simple, imparfait, conditionnel présent
3. Règles d’accord du participe passé (comme avec être)
4. Agent : par (action volontaire) / de (état, sentiment, origine)
5. Omission de l’agent quand inconnu ou non important
6. Montrer 12–15 exemples (actualité, instructions, descriptions formelles)
7. Exercice guidé : transformer 10 phrases actives → passives
8. Production : réécrire un court texte actif au passif (8–10 phrases)

**Tableau formation voix passive (exemples clés):**

| Temps                  | Exemple actif                        | Exemple passif                              |
|------------------------|--------------------------------------|---------------------------------------------|
| Présent                | On vend les billets.                 | Les billets sont vendus.                    |
| Passé composé          | Ils ont construit la maison.         | La maison a été construite.                 |
| Imparfait              | On réparait la route.                | La route était réparée.                     |
| Futur simple           | On ouvrira le musée.                 | Le musée sera ouvert.                       |
| Conditionnel présent   | On devrait interdire cela.           | Cela devrait être interdit.                 |

### Vocabulary Set (24 words)
**Semantic Domain:** Actualités, instructions, processus, formalités, événements publics
être construit, être réparé, être interdit, être autorisé, être vendu, être acheté, être fermé, être ouvert, être annoncé, être publié, être élu, être nommé, être invité, être accusé, être félicité, être puni, être détruit, être créé, être organisé, être annulé, être reporté, par les autorités, par le gouvernement, par l’entreprise, par les employés

### Speaking Scenario
**Domain:** Décrire un événement d’actualité ou un processus
Explique un événement récent (inauguration, accident, élection, fermeture d’usine…) en utilisant la voix passive au moins 6 fois (focus sur l’action, pas sur qui l’a fait).

### Reading/Listening Component
**Type:** Article de presse court + audio (140–170 secondes)
« Un nouveau musée a été inauguré à Istanbul » (texte avec 8–10 passives)
→ 9 questions (identifier voix passive, transformer en actif, vrai/faux, reformuler)

### Homework Assignment
**Type:** Audio
Enregistre-toi (90–120 secondes) : décris un événement récent (dans ta ville, au travail, dans les news) en mettant l’accent sur ce qui a été fait / décidé / construit / annulé…
Utilise **au moins 7 voix passive** (différents temps).

**Rubric:**
- [ ] Voix passive correcte ≥ 7 fois (formation + accord)
- [ ] Au moins 3 temps différents utilisés
- [ ] ≥ 10 mots de la liste intégrés correctement
- [ ] Discours clair, structuré, registre neutre/formel
- [ ] Prononciation et fluidité B1.2 niveau
**Pass:** 4/5 critères remplis

### Assessment Checkpoint
**Mini-Quiz (7 questions):**
1. Transforme : Les ouvriers réparent le pont. → Le pont ……… .
2. Passé composé : On a fermé l’aéroport. → L’aéroport ……… .
3. Futur : On va construire un hôpital. → Un hôpital ……… .
4. Traduis : “La loi a été votée hier.” → La loi a été votée hier.
5. Agent : Ce livre est écrit ……… un auteur célèbre. → par
6. Omission : Les fenêtres sont nettoyées tous les jours. → (agent omis – pourquoi logique ?)
7. Conditionnel : Il faudrait interdire les plastiques. → Les plastiques ……… .
