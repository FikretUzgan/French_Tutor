## Week 16 / Block 4 / CEFR Level A2.2

### Learning Outcomes (CEFR Descriptors)
- Can use reflexive and reciprocal verbs to describe daily routines, emotions and reciprocal actions in present and past tenses.
- Can understand short texts and conversations containing advanced reflexive structures.
- Can produce coherent short texts describing habits, changes in state and mutual actions.

### Interaction Tier
**Tier:** 1→2
**Interaction Type:** Semi-Initiated

### Grammar Target
**Form:** Verbes pronominaux avancés (routines, changements d’état, réciproques, sens passif / impersonnel) + accord du participe passé
**Complexity Rating:** 6/10
**Prerequisite Knowledge:** Verbes pronominaux de base (se laver, se lever), passé composé avec être, accord du participe
**Scaffolding Steps:**
1. Rappel : position du pronom + accord avec être (se lever → je me suis levé(e))
2. Usages avancés :
   - routine / habitude (se réveiller tard, s’habiller vite)
   - changement d’état (se fâcher, s’énerver, se calmer)
   - réciproque (s’aimer, se parler, se voir souvent)
   - sens passif / impersonnel (se vendre bien, ça se fait)
3. Tableau conjugaison + accord exemples
4. 12–15 exemples contextuels variés
5. Exercice guidé : gap-fill + choisir accord (masculin/féminin/pluriel)
6. Transformation : 8 phrases sans pronominal → avec verbe pronominal
7. Production : 7–8 phrases sur routine + émotions + relations

**Tableau accord participe passé (être):**
- Je me suis lavé(e) → accord avec sujet (e si féminin)
- Nous nous sommes vus → accord pluriel
- Elles se sont parlé(e)s → accord réciproque (pluriel + e si féminin)

**Verbes clés à maîtriser :** se réveiller, s’habiller, se coucher, s’énerver, se calmer, se disputer, se réconcilier, s’entendre, se voir, se téléphoner, se souvenir, s’intéresser à, se passer

### Vocabulary Set (22 words)
**Semantic Domain:** Émotions, relations, routine avancée, changements d’humeur
se fâcher, se disputer, se réconcilier, s’entendre bien/mal, s’énerver, se calmer, se souvenir de, s’intéresser à, se passer (ça s’est bien passé), se produire, se lever tôt/tard, s’habiller vite, se coucher tard, se reposer, se détendre, se stresser, se fatiguer, se sentir bien/mal, se voir souvent, se téléphoner, se rencontrer, s’embrasser

### Speaking Scenario
**Domain:** Parler de sa relation / routine émotionnelle
Raconte comment se passe ta relation avec un ami / ta famille / ton partenaire (vous vous disputez ? vous vous entendez bien ? comment vous vous réconciliez ?). Utilise au moins 6 verbes pronominaux différents.

### Reading/Listening Component
**Type:** Court texte + audio (110–140 secondes)
« Une dispute et une réconciliation » (histoire courte avec 7–8 verbes pronominaux)
→ 8–9 questions (remplacer par verbe pronominal, vrai/faux, accord du participe, compléter)

### Homework Assignment
**Type:** Audio
Enregistre-toi (70–100 secondes) : raconte une journée typique + une situation émotionnelle récente (vous vous êtes disputés ? tu t’es énervé(e) ? comment ça s’est passé ?).
Utilise **au moins 6 verbes pronominaux** (dont 3 au passé composé avec accord correct).

**Rubric:**
- [ ] ≥ 6 verbes pronominaux utilisés
- [ ] Accord du participe passé correct dans au moins 3 cas au passé composé
- [ ] ≥ 10 mots de la liste utilisés correctement
- [ ] Prononciation claire, rythme naturel, histoire compréhensible
**Pass:** 4/4 critères remplis

### Assessment Checkpoint
**Mini-Quiz (7 questions):**
1. Conjugue au passé composé : Nous ……… (se disputer) hier soir. → nous nous sommes disputés
2. Accord : Elle ……… (se lever) tard. → s’est levée
3. Réciproque : Ils ……… (se voir) tous les jours. → se voient
4. Traduis : “Nous nous sommes réconciliés rapidement.” → Nous nous sommes réconciliés rapidement.
5. Choisis : Ça ……… bien ……… hier ? → s’est / passé
6. Erreur : Je me suis intéressé à ce livre. → correction si féminin ?
7. Question : Vous ……… souvent ………-vous ? → vous / téléphonez
