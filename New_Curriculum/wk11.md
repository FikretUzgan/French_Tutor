## Week 11 / Block 3 / CEFR Level A2.1

### Learning Outcomes (CEFR Descriptors)
- Can describe future plans and intentions in simple sentences.
- Can make simple predictions about the near future.
- Can express hopes, wishes and basic probabilities using future forms.

### Interaction Tier
**Tier:** 1→2 transition
**Interaction Type:** Reactive → very simple semi-initiated

### Grammar Target
**Form:** Futur simple (verbes réguliers + irréguliers courants : aller, faire, être, avoir, venir, vouloir, pouvoir)
**Complexity Rating:** 4/10
**Prerequisite Knowledge:** Présent des verbes réguliers, infinitif, terminaisons -ai/-as/-a/-ons/-ez/-ont
**Scaffolding Steps:**
1. Règle de base : infinitif + terminaisons du futur (-ai, -as, -a, -ons, -ez, -ont)
2. Tableau de conjugaison pour verbes réguliers (-er, -ir, -re)
3. Irréguliers les plus utiles : aller → ir-, faire → fer-, être → ser-, avoir → aur-, venir → viendr-, vouloir → voudr-, pouvoir → pourr-
4. Montrer 10–12 exemples concrets (projets personnels, météo demain, vacances, week-end)
5. Exercice guidé : conjuguer 8 verbes au futur + gap-fill
6. Transformation : 5 phrases au présent → futur simple
7. Production : écrire 6–7 phrases sur ses projets pour les prochaines semaines/mois

**Tableau Futur simple (parler & aller):**
parler
je parlerai    tu parleras    il/elle parlera    nous parlerons    vous parlerez    ils/elles parleront
aller
j’irai         tu iras        il/elle ira        nous irons        vous irez        ils/elles iront

### Vocabulary Set (24 words)
**Semantic Domain:** Projets futurs, vacances, météo prévisionnelle, intentions
demain, après-demain, bientôt, la semaine prochaine, le mois prochain, l’année prochaine, cet été, cet hiver, je vais, je voudrais, peut-être, probablement, certainement, j’espère, je pense que, je crois que, voyager, visiter, découvrir, apprendre, réussir, partir, arriver, rester, revenir, prendre des vacances

### Speaking Scenario
**Domain:** Projets de vacances / week-end
Raconte ce que tu vas faire pendant tes prochaines vacances ou le week-end prochain (minimum 6–7 phrases au futur simple).

### Reading/Listening Component
**Type:** Court texte + audio (100–130 secondes)
« Mes projets pour 2026 » (une personne parle de ses plans)
→ 7–8 questions de compréhension (oui/non, vrai/faux, réponse courte)

### Homework Assignment
**Type:** Texte
Écris 10 phrases sur tes projets pour les 6–12 prochains mois.
Utilise le **futur simple** au moins **7 fois**.

**Rubric:**
- [ ] Futur simple correct ≥ 7 fois
- [ ] ≥ 8 mots de la liste utilisés correctement
- [ ] Phrases reliées avec des mots de liaison simples (et, mais, parce que, quand, si…)
- [ ] Texte compréhensible même avec de petites erreurs
**Pass:** 4/4 critères remplis

### Assessment Checkpoint
**Mini-Quiz (6 questions):**
1. Conjugue : Nous ……… (aller) à Paris en juillet. → irons
2. Irrégulier : Je ……… (faire) du sport tous les jours. → ferai
3. Traduis : “Il fera froid demain.” → Il fera froid demain.
4. Question : Où ………-tu ……… vacances cet été ? → passeras / tes
5. Négation : Ils ……… pas ……… (venir) avec nous. → viendront
6. Choisis : Tu ……… (pouvoir) m’aider demain ? → pourras
