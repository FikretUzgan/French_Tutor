## Week 26 / Block 7 / CEFR Level B2.1

### Learning Outcomes (CEFR Descriptors)
- Can use participle clauses (gérondif, participe présent, participe passé, absolute constructions) to compress sentences and achieve stylistic elegance.
- Can distinguish between gérondif (adverbial), participe présent (adjectival/verbal) and participe passé in context.
- Can rewrite longer sentences into more concise, literary or formal versions using participles.

### Interaction Tier
**Tier:** 3
**Interaction Type:** Fully Initiated (student drives conversation, AI acts as peer for literary / stylistic analysis)

### Grammar Target
**Form:** Participle Clauses (gérondif, participe présent, participe passé, absolute constructions) + Sentence Compression
**Complexity Rating:** 8/10
**Prerequisite Knowledge:** Present participle formation, relative clauses, conjunctions, complex sentence structure
**Scaffolding Steps:**
1. Clear distinction:
   - Gérondif (en + -ant): simultaneous action, manner, cause → En travaillant, elle écoute de la musique.
   - Participe présent (-ant): verbal adjective or noun modifier → Une femme travaillant dur…
   - Participe passé: adjective or reduced relative clause → Le livre publié récemment…
   - Absolute construction: Le travail terminé, nous sommes partis.
2. Show 12–15 progressive examples (daily → formal → literary)
3. Guided practice: identify function (manner / cause / simultaneity / adjective)
4. Compression exercise: rewrite 8 long sentences using participles
5. Style comparison: same content in simple vs compressed / elegant version
6. Production: rewrite a 120–150 word text using at least 5 different participle constructions

### Vocabulary Set (20 words)
**Semantic Domain:** Literary style, elegance, cause/manner/simultaneity, formal writing

**Domain:** Stylistic / literary discussion

Rewrite it using **at least 6 different participle constructions** (mix gérondif, participe présent, passé, absolute) to make it more concise and stylistically elegant.

**Rubric:**
- [ ] ≥ 6 distinct participle constructions used correctly
- [ ] Improved conciseness and flow clearly visible
- [ ] Grammatical accuracy and stylistic appropriateness (B2.1 level)
- [ ] ≥ 10 vocabulary items from the week integrated

**Pass:** 4/5 criteria met

### Assessment Checkpoint
**Mini-Quiz (7 questions):**
1. Transform: Comme elle travaillait, elle écoutait de la musique. → En travaillant…
2. Function: “La lettre écrite par Paul est arrivée.” → participe passé as reduced relative clause
3. Compression: Parce que le travail était terminé, nous sommes partis. → Le travail terminé, nous sommes partis.
4. Choose correct: Une équipe ……… dur a gagné. → travaillant
5. Error: En étant fatigué, il a continué. → correction?
6. Gérondif or participe présent? “En marchant rapidement, il évitait la pluie.” → gérondif (manner)
7. Absolute construction: La nuit ……… , les étoiles brillaient. → tombée
