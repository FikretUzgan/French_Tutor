## Week 13 / Block 4 / CEFR Level A2.2

### Learning Outcomes (CEFR Descriptors)
- Can make polite requests and suggestions using conditional forms
- Can express hypothetical situations in a simple way
- Can use basic politeness strategies in everyday situations

### Interaction Tier
**Tier:** 1→2
**Interaction Type:** Semi-Initiated (student can now ask very basic questions, AI still guides)

### Grammar Target
**Form:** Conditionnel présent (formation + the two most common uses:
1. polite request
2. hypothesis / polite suggestion)
**Complexity Rating:** 5/10
**Prerequisite Knowledge:** Should understand futur simple stems + imparfait endings logic
**Scaffolding Steps:**
1. Review formula: futur stem + imparfait endings (same endings: -ais, -ais, -ait, -ions, -iez, -aient)
2. Show a table with the most common verbs (vouloir, pouvoir, devoir, aimer, venir, aller)
3. Polite request examples: Je voudrais…, Pourriez-vous…, Ça vous dérangerait si…
4. Polite suggestion / hypothesis: Ce serait bien si…, Je pense que tu devrais…
5. Gap-fill → 8 sentences
6. Write 5–6 original polite sentences

**Example Table (core verbs):**

### Vocabulary Set (18 words)
**Semantic Domain:** Polite requests, gentle suggestions, daily shopping / services
je voudrais, s’il vous plaît, merci beaucoup, pardon de vous déranger, est-ce que vous pourriez, ça vous ennuierait si, je vous conseille de, ce serait gentil si, un café s’il vous plaît, l’addition s’il vous plaît, je peux payer par carte ?, c’est possible de…, désolé(e), pas de problème, avec plaisir, volontiers, je préférerais, ça dépend

### Speaking Scenario
**Domain:** Cafe / restaurant
You are speaking with a waiter. You order, ask for something, request the bill, and thank politely.

### Reading/Listening Component
**Type:** Short dialogue (audio + text) – 90–110 seconds
Cafe waiter-customer dialogue (3 requests + 1 polite refusal / suggestion)
→ 7–8 simple comprehension questions (mostly true/false + short answer)

### Homework Assignment
**Type:** Audio
Imagine yourself as a customer and record 60–80 seconds:
Start a short dialogue in a cafe/restaurant with 4–5 polite requests (the waiter will not answer; you speak alone).
Use at least 4 conditionnel present forms.

**Rubric:**
- [ ] Conditionnel present ≥4 correct uses
- [ ] At least 6 words/phrases from the list used correctly
- [ ] Pronunciation is clear (meaning remains intact)
- [ ] Polite tone is clear (intonation, please/thanks usage)

**Pass:** At least 3 of 4 criteria met

### Assessment Checkpoint
**Mini-Quiz (6 questions):**

1. Conjugue: Je (vouloir) un verre d’eau, s’il vous plaît.
   → je voudrais

2. Traduis : “Est-ce que vous pourriez m’apporter le menu ?”
   → Is the conditional used correctly?

3. Choisis la forme correcte : Tu … venir avec moi ce soir ? (voudrais / veux)

4. Transforme en conditionnel poli : Je veux parler au manager.
   → Je voudrais parler au manager.

5. Trouve l’erreur : Je aimerais un café crème.
   → What is the correct form?

6. Complète : Ça vous … si je fume ici ? (dérangerait)
